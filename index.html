Code

<script src="http://coincalendar.eu/widget/coin.js"></script>


	<script>
		CoinWidgetCom.go({
			/* make sure you update the wallet_address or you will not get your donations */
			wallet_address: "9WkrThJKCwwqUC8oL4HGihmDUSVvuEhaUU"
			, currency: "noblecoin"
			, counter: "count"
			, alignment: "bl"
			, qrcode: true
			, auto_show: false
			, lbl_button: "Donate NobleCoin"
			, lbl_address: "Donate NobleCoin to this Address:"
			, lbl_count: "donations"
			, lbl_amount: "NOBL"
		});
	</script>

Settings

currency [noble] - This widget supports only Noble Coin.
counter [amount] - amount will show the total amount received in NOBL. Hide will hide the counter as well as hide the transaction count and total amount received within the window.
qrcode [true/false] - Set to true if you want to show the QR code generator that appears at the bottom left of the window. Set to false to hide the QR code icon.
auto_show [true/false] - Set to true if you want the window to auto appear as soon as the counter finishes loading.
